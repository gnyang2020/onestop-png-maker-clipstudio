쥐냥작가 가이드에 그린 PNG를 넣으면, 칸마다 잘라서 카카오·오지큐 규격에 맞는 png로 자동 저장돼요.

🟢 일반 모드 · 카카오 / 오지큐 (각각 선택)
  • 카카오용 PNG — 글자에 테두리를 이미 입힌 버전을 넣어요.
  • 오지큐용 PNG — 테두리 없는 원본을 넣으면, 도구가 흰 테두리를 자동으로 둘러줘요.
  • 둘 중 하나만 넣어도 되고, 둘 다 넣으면 한 번에 양쪽 다 만들어져요.
🟡 카카오 — 36칸 중 특별한 칸
• 36번 칸 = 아이콘(78×78)으로 자동 변환
• 35번 칸 = 카카오에선 제외 (오지큐 메인 전용 칸이라서)
🟢 오지큐 — 칸별 자동 변환
• 1~34번 = 일반 스티커 740×640
• 35번 = 메인 240×240
• 36번 = 탭 96×74
• 흰 테두리 두께 — 일반 스티커 8px · 메인 3px · 탭 1px (스티커 두께는 설정에서 조절 가능)
🔵 미니 모드 · 카카오 전용
  • 카카오용 PNG 1장만 넣으면 돼요.
  • 그림이 있는 칸만 골라 180×180 크기로, 번호를 붙여 자동 저장해요.
  • 반드시 쥐냥작가의 가이드에 그린 PNG만 사용하세요. (칸 위치가 맞아야 정확히 잘려요)
  • 가이드선·배경처럼 필요 없는 레이어는 눈을 끄거나 삭제한 뒤 PNG로 저장하세요.
  • 카카오는 테두리를 입힌 버전, 오지큐는 테두리 없는 원본 — 헷갈리지 않게 구분해서 넣으세요.
  • 안 그린 빈 칸은 자동으로 제외되니, 비워둬도 괜찮아요.
